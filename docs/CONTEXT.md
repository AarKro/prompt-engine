# Product Name: Prompt Engine

## Product Purpose
Users can create customized, optimized prompts in a few steps and copy them with one click to use them directly in other LLM tools (ChatGPT, Claude, etc.) – without technical knowledge or time-consuming optimizations.

## Product Goal
To eliminate the dissonance between fast and good prompts: Users should be able to create prompts that are time-saving AND high quality at the same time.


## Why this works

### Resolving the core problem:
Before: Users have to choose: Fast prompts = bad, Good prompts = time-intensive
After: The Prompt Wizard takes over the optimization, users save time AND receive better results

### Concrete application:

* User selects their use case (e.g., "Write product description")
* The Wizard automatically generates an optimized prompt template
* User copies the finished prompt → pastes it into ChatGPT/Claude

Result: High-quality output in minimal time
